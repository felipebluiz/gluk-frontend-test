# Development

Para instalar todas as dependências, execute o comando:

```bash
$ npm install
```

Levantar a aplicação em modo de desenvolvimento:

```bash
$ npm run dev
```

# Produção

Gerando uma `build` para produção:

```bash
$ npm run build
```

Rodando o ambiente em modo de produção:

```bash
$ npm run preview
```
